# Day 01 Notes

## Terminal Commands

### pwd
Displays the current directory I am working in.

### ls
Lists the files and folders in the current directory.

### cd
Changes from one directory to another.

### mkdir
Creates a new directory.

### touch
Creates a new empty file.

---

## Git Commands

### git init
Creates a new Git repository.

### git add
Stages files so they are ready to be committed.

### git commit
Saves the staged changes with a descriptive message.
